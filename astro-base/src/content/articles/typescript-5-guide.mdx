# TypeScript 5.0 완벽 가이드

TypeScript 5.0이 출시되었습니다. 이번 버전에서는 많은 새로운 기능과 개선사항이 추가되었습니다.

## 주요 기능

### 1. Decorators

ECMAScript 표준을 따르는 새로운 데코레이터가 추가되었습니다.

<Picture src={photo} formats={['avif', 'webp']} alt="별밤" />

```typescript
function logged(target: any, context: ClassMethodDecoratorContext) {
  const methodName = String(context.name);

  function replacementMethod(this: any, ...args: any[]) {
    console.log(`Calling ${methodName}`);
    const result = target.call(this, ...args);
    console.log(`Called ${methodName}`);
    return result;
  }

  return replacementMethod;
}

class Calculator {
  @logged
  add(a: number, b: number) {
    return a + b;
  }
}
```

### 2. const 타입 매개변수

```typescript
function identity<const T>(value: T): T {
  return value;
}

const arr = identity([1, 2, 3]); // readonly [1, 2, 3]
const obj = identity({ x: 10 }); // { readonly x: 10 }
```

### 3. 성능 개선

TypeScript 5.0은 이전 버전 대비 최대 **3배 빠른** 컴파일 속도를 자랑합니다.

## 실전 예제

### Before (TypeScript 4.x)

```typescript
type Result =
  | {
      data: string;
      error: null;
    }
  | {
      data: null;
      error: Error;
    };

function processResult(result: Result) {
  if (result.data) {
    return result.data.toUpperCase();
  }
  throw result.error;
}
```

### After (TypeScript 5.0)

```typescript
type Result<T, E = Error> = { ok: true; value: T } | { ok: false; error: E };

function processResult<T>(result: Result<T>) {
  if (result.ok) {
    return result.value;
  }
  throw result.error;
}
```
