# Tailwind로 디자인 시스템 구축하기

디자인 시스템은 일관된 사용자 경험을 제공하는 핵심 요소입니다.

## CSS 변수 기반 토큰 시스템

### 색상 시스템

```css
:root {
  --background: #ffffff;
  --foreground: #1a1a1a;
  --primary: #1a1a1a;
  --secondary: #f5f5f5;
  --muted: #6b6b6b;
  --border: rgba(0, 0, 0, 0.2);
}

.dark {
  --background: #0f0f0f;
  --foreground: #ededed;
  --primary: #ededed;
  --secondary: #262626;
  --muted: #a3a3a3;
  --border: rgba(255, 255, 255, 0.2);
}
```

### Tailwind 통합

```javascript
// tailwind.config.js
export default {
  theme: {
    extend: {
      colors: {
        background: 'var(--background)',
        foreground: 'var(--foreground)',
        primary: 'var(--primary)',
      },
    },
  },
};
```

## 타이포그래피 스케일

@layer base {

## 성능 최적화

1. **Purge 설정** - 사용하지 않는 CSS 제거
2. **JIT 모드** - 필요한 클래스만 생성
3. **CSS 변수** - 다크모드 토글 최적화

## 결론

Tailwind CSS와 CSS 변수를 조합하면 확장 가능하고 유지보수하기 쉬운 디자인 시스템을 만들 수 있습니다.
